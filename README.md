# ChargeLog

iOS app for logging electric vehicle charging sessions.

Track charge sessions with start/stop times, energy (kWh), cost, and odometer. Mark sessions as paid or open to manage reimbursement. Export data as CSV.

## Features

- Log charging sessions with station, energy, price, and odometer
- Start a session now and stop it later
- Payment status tracking (open / paid)
- Filter history by all / open / paid
- Station autocomplete from previous entries
- CSV export

## Tech

- Swift, SwiftUI, SwiftData
- iOS 26+
